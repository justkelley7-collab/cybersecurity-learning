# Log Analysis

This repository contains hands-on exercises focused on interpreting
system and security logs to identify anomalies, troubleshoot issues,
and support basic incident investigation.

## Overview
The exercises in this repository simulate real-world scenarios where
log data is used to understand system behavior and detect potentially
malicious activity.

## Skills Demonstrated
- System and security log interpretation
- Identification of suspicious authentication activity
- Basic incident investigation and reporting
- Log-based troubleshooting

## Log Sources Covered
- Linux system and authentication logs
- Windows security event logs
- Web server access and error logs

## Purpose
This repository is intended for learning and demonstration purposes.
All log files are simulated or sanitized and do not contain real
production data.
